# vue-modules-test

## 介绍
> 测试各种第三方modules

## 使用
1. 在views创建一个这个库的名字的文件夹，在里面创建一个index.vue作为主入口
2. 在`router/index.js`中加入这个路由，路由名为这个库的名字
3. 关于此库的各种引入，或者组件请在这个文件夹内执行，免得以后找不到了
