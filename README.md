# vue-modules-test

## 介绍

> 测试各种第三方 modules

## 使用

1. 在 views 创建一个这个库的名字的文件夹，在里面创建一个 index.vue 作为主入口
2. 系统将自动引入此文件夹，并且在页面上会出现这个文件夹名称的选项，点击后会显示 index.vue 内容
3. 页面上名称的顺序将根据 views 内文件夹的顺序排列
4. 关于此库的各种引入，或者组件请在这个文件夹内执行，免得以后找不到了
